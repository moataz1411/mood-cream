**Mood Cream**

Turn ur mood into ice cream flavpr!!!!

Mood Cream is a fun website that matches your mood with an ice cream flavor according to my mum 😂. Pick how u are feelings, and the website will reveal a flavor, a quote, and a colorful theme that fits ur vibe.

**why cream mode**

My mum always told me that every ice cream flavor expresses a certain mood. So I decided to turn that idea into a fun website!

**features**

- choose ur mood.
- get ur ice cream flavor.
- my street ice cream truck audio.
- about me page.
- fun quotes and emoji for each mood.

**languages used**

- HTML
- CSS
- JavaScript
